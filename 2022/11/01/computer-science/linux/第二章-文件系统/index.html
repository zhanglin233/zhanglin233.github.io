# 文件系统

# 目录操作命令

# 创建目录

mkdir 命令：创建目录，其后的命令参数可以是目录的绝对路径和相对路径，以及多个目录。

# 删除目录

rmdir 命令：删除目录。rmdir 命令的执行条件是用户在要被删除目录的父母录下具有写权限，并且被删除目录是空目录（即除。和。。 意外没有其他文件）

# 修改该当前目录

cd 命令：其后的命令行参数可以是目录的相对路径和绝对路径，默认的状态下返回用户主目录

# 查看当前目录

pwd 命令：查看当前目录

# 列出指定目录

ls 命令：列出指定目录。用户在任何一个目录下执行 ls -l, 就能将当前目录中的所有文件、子目录、符号链接等一列表的方式显示出来，同时还可以借助 ls -l 命令了解到文件类型、权限等一些重要的文件属性。

可以看出显示内容分为两大部分，第一部分是文件属性，第二部分是文件名（当然也包括子目录名）。如果继续对文件属性进行划分，那么最左边的是文件类型；其次是三个权限域；在权限域之后是文件硬链接数或目录的子目录数；剩下的四个域分别是文件所有者、文件所有者所在组、文件大小（以字节为单位）以及文件最后修改时间。

Linux 操作系统有一个特点，那就是 “一切都是文件”，所有资源都尽可能映射成文件来管理（网络设备例外）。

标识 文件类型
- 普通文件
d 目录文件
c 字符设备文件
b 块设备文件
p 管道文件或 FIFO 文件
l 符号链接文件
s 套接字文件

Linux 操作系统还对文件实施完善的访问权限管理吗，普通用户要对某个文件进行相关操作，必须具备对该文件的对应操作权限。

标识 文件访问权限
r 读权限
w 写权限
x 执行权限

# 文件操作命令

# 显示文件内容

  • cat 命令：显示文件内容，其后的命令行参数（文件）依次被打开，文件内容全部显示在屏幕上。

  • more 命令：逐屏显示文件内容。与 cat 命令不同，more 命令并不是一次性显示完。若文件长度大于 23 行，屏幕底部会显示 “More “或” More + 已显示百分比 “。

# 复制文件

  • cp 命令：复制命令，基本格式如下：

    cp [-i][-r] oldfile newfile

    执行命令前，若 newfile 文件存在，则 oldfile 内容直接覆盖 newfile 的内容；若 newfile 不存在，则新建 newfile 文件并进行复制。

    该命令选项有两个，- 表示请求确认；-r 表示复制目录及其内部的所有文件到新目录下。

# 移动文件

  • mv 命令：给文件换名，或将文件移至新的目录下。

    mv oldfile newfile

# 删除文件

  • rm 命令：删除文件，其格式如下：

    rm [-i] [-r] [file1 file2 file3 ···]

​ 成功执行该命令将永久删除改文件。

​ 选项 - i 表示删除文件前需用户确认；-r 表示强制删除文件，包括目录文件。

# 打印文件

  • lp 命令：打印命令，其基本格式如下：

    lp [-operation] file [file1 file2 ···]

​ 成功执行该命令并连接可操作打印机后，打印任务交予后台，屏幕将出现 Shell 的命令提示符，可继续前台的工作。

选项 具体功能
-d 在指定的打印机上执行打印任务
-m 打印任务完成，通过 mail 方式告知用户
-n 指定打印份数
-s 取消屏幕的提示信息
-t 文档封面打印标题
-w 打印完成后在用户端显示提示信息

# 格式化输出文件

  • pr 命令：格式化输出命令，，将文件内容以标准打印格式显示在屏幕上。

# 链接文件

  • ln 命令：链接文件命令，给一个已经存在的文件再取一个名字，新的文件名指向旧的文件名，文件内容并没有被复制。

# 基本实用程序

  • wc 命令：文件统计命令，可以同时对一个或者多个文件进行行、单词和字符数进行统计。
选项 具体功能
-i 只对行进行计数
-w 只对单词进行计数
-c 只对字符数进行计数
  • find 命令：查找文件命令，在指定目录下递归查找指定文件。
find pathname1 [pathname2 ···] option1 [option2 ···] [action]

pathname 是指定目录的路径，option 是所需查找的文件部分属性，action 是找到文件后的动作。

  • sort 命令：文件内容排序命令，通过逐行对比内容的 ASCII 码的大小实现排序

  • grep 命令：文件内容搜索命令，在一个或者多个文件中搜索包含指定字符串的行并将其显示出来。

  • head 命令： 显示文件头部信息，缺省时默认为前 10 行。如果给定一个减号开始的数字，则显示文件开头的指定行数

    head filename
head -5 filename # 显示开头5行

  • tail 命令：显示文件尾部信息，缺省时默认为后 10 行。如果给定一个减号开始的数字，则显示文件结尾的指定行数。

  • cut 命令：切取文件内容命令，通过 “切取” 文件中的行或字段，达到显示部分内容的目的。

    选项 具体功能
    -f 指定字段的位置
    -c 指定列的位置
    -d 指定字段分隔符，缺省时默认为 tab

# 文件的权限管理

# 文件的权限种类

Linux 下可以对三类用户进行权限限制

  • 用户 (user)，简称 u， 也就是文件所有者；
  • 用户组 (group)，简称 g，也就是和文件所有者同组的用户；
  • 其他用户 (other)，简称 o，也就是除用户、用户组以外的所有用户。
  • 这些用户加起来叫做所有用户，简称 a。只有文件所有者可以改变文件的权限，用户的权限高于用户组的权限；而超级用户永远拥有最高权限，不受权限控制。

​ Linux 下权限表示的形式有两种，字母形式和数字形式。对于字母，r、w、x 分别为 read、write、execute (执行，对于目录是进入的权限)。对于数字，r=4,w=2,x=1.

​ 例如某权限为 - rwxr-----, 第一个字符表示具有此权限的实体时目录还是文件，如果是目录则为 d，如果是文件则为 -。

# 文件权限的检测与修改

  • chmod 命令：修改文件或目录的访问权限，基本格式如下

    chmod pattern file

    pattern 有两种表示形式：字母形式和数字形式。前者由用户类别（u、g、o）、如何改变（+、-）和权限（r、w、x）3 部分组成；后者把 9 位字母形式的权限划分为 3 组，由 3 位 8 进制数表示

    chmod u+x test1
chmod 644 test2

# 文件所有权的修改

  • chown 命令：修改文件或目录的所有者命令。-R 表示递归地改变指定目录及其下所有子目录和文件的所有者；-v 表示显示 chown 命令所做的工作。
  • chgrp 命令：修改文件或目录的用户组命令。

# 文件名置换

通配符 含义
？ 表示匹配任意单个字符
* 表示匹配任意多个字符
[list] 表示匹配 lsit 中的任意单个字符
[!list] 表示匹配不在 list 中的任意单个字符
